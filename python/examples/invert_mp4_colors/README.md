# MP4 color inversion demo

This is a simple demo showcasing how to invert colors of an MP4
with Boombox.

## Prerequisites

To run the demo you'll need `boomboxlib`, which can be installed like that:

```bash
pip install ../..
```

And some player, like `vlc` or `ffplay` to play the created MP4 file.

## Usage

1. Run the python program

```bash
python main.py
```

2. Play the output MP4:

```bash
ffplay inverted.mp4
```
